# TRACE_OPEN

TRACE_OPEN is a [gadget from Inspektor
Gadget](https://inspektor-gadget.io/).

## How to use

```bash
$ export IG_EXPERIMENTAL=true
$ sudo -E ig run ghcr.io/CHANGEME-ORG/CHANGEME-GADGET-NAME:latest
```

## Requirements

- ig v0.25.0 (CHANGEME)
- Linux v5.15 (CHANGEME)
